scaling-lana
============

A Google App Engine web application to keep a score at volleyball games and keep stats on players.

## Technology stack
- [Google App Engine][1]
- [Angular JS][4]
- [Maven 3.1.0][9]
- [Firebase][15]
- [Bootstrap][5]
